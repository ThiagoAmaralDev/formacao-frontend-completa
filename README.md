# 📚 Formação Front-end: HTML, CSS, JavaScript, React e +

Este repositório foi criado para documentar minha jornada de aprendizado no curso do Matheus Battisti, hora de codar. Meu objetivo aqui é ir além do conteúdo das aulas, aplicando os conceitos em projetos autorais logo após finalizar as versões guiadas.

---

## 🛠️ Tecnologias em Estudo

- Linguagens: HTML5, CSS3, JavaScript (ES6+), TypeScript
- Biblioteca Principal: React.js
- Ferramentas: Git, GitHub, VS Code

---

### 📂 Estrutura do Repositório

O repositório está dividido em módulos. Dentro de cada módulo, você encontrará duas pastas principais:

1. guiado/: Contém o projeto exatamente como desenvolvido durante as aulas do curso, seguindo o passo a passo do instrutor.
2. autoral/: Contém um projeto novo, criado por mim do zero, utilizando a mesma base tecnológica do projeto guiado, mas com funcionalidades extras ou uma temática diferente.
